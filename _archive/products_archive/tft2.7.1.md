 # TFT v2.7.1

Service Improvements related to ThreeFold Token

## Components

- [TFT](https://github.com/threefoldfoundation/tft)
- [TFT-Stellar](https://github.com/threefoldfoundation/tft-stellar)
- [BSC Bridge](https://github.com/threefoldtech/binance-chain-bridge-dapp)

## Features

- TFT implemented as a cross-chain asset (BToken) on the [Binance Smart Chain (BSC)](https://www.binance.org/en/smartChain).
- Implemented support TFT buying with BTC
- Improvements to the token statistics service
- Implemented TFT vesting service
